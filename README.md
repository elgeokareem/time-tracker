# time-tracker

Time tracker to know in what activities I'm spending my time on.

pnpm dev  
pnpm build  
pnpm preview.  

Using: Vite, React, Zustand, Tailwind.

## Want to use npm instead of pnpm?  
Just delete pnpm-lock.yaml file and run npm install then run

npm run dev  
npm run build  
npm run preview.
